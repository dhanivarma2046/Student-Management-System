# Student Management System

A desktop-based Student Management System built using Python, Tkinter, and SQLite.

## Features
- Add student records
- Search students
- Update student details
- Delete student records
- Clear input fields
- Show all students
- Export data to CSV
- Display records using Treeview
- SQLite database integration

## Technologies Used
- Python
- Tkinter
- SQLite
- CSV

## How to Run
1. Install Python.
2. Clone or download this repository.
3. Run the file:

```bash
python main.py
